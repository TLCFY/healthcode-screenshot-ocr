## Package-Installation

1. 安装`tesseract-ocr`及其中文包

    ```shell
    sudo apt install tesseract-ocr
    sudo apt install tesseract-ocr-chi-sim
    ```

2. 安装百度官方`OCR`依赖包`baidu-aip`

    ```shell
    pip install baidu-aip
    ```
